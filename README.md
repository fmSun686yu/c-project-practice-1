# c-project-practice-1

## 📌 仓库简介

本仓库用于记录我在学习《C语言项目开发实战入门》一书过程中完成的示例程序与项目实践。

> 教材详细信息：
> - 书名：C语言项目开发实战入门
> - 作者：明日科技 编著
> - 出版发行：吉林大学出版社
> - ISBN：978-7-5677-8741-4

在跟随书中教程编写示例代码的基础上，我对部分程序进行了修改、扩展与优化，并加入了自己的理解与创新，旨在巩固 C 语言基础知识，逐步提升编程能力与工程实践能力。

本仓库既是学习过程的记录，也是 C 语言实战能力的持续积累。

## 🎯 学习目标

通过本仓库的持续维护，希望达到以下目标：

- 系统掌握 C 语言的基础语法与核心概念  
- 理解常见 C 语言程序与项目的设计思路  
- 学会将零散的知识组织为可运行的程序或小型项目  
- 培养良好的代码风格与基本的工程结构意识  
- 在原有示例基础上进行思考、修改与创新，而非机械照抄

### 目录说明

- **`docs/`**  
  用于记录学习过程中的笔记、总结与反思，帮助加深理解与回顾知识。

- **`projects/`**  
  相对完整的 C 语言项目，逐步引入模块化与简单工程结构。

- **`experiments/`**  
  用于验证概念、测试想法的实验性代码，如指针、内存布局等相关探索。

- **`.vscode/`**  
  存放 VSCode 的 json 配置文件，通常已在 `.gitignore` 中忽略。

- **`build/`**  
  存放编译生成的可执行文件或中间产物，通常已在 `.gitignore` 中忽略。
