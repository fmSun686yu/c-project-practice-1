# 1-Tetris

## 项目简介

本项目跟随教材，设计一个趣味俄罗斯方块游戏。

## 项目结构说明

> 1-Tetris/                                             <br>
> ├── README.md         // 项目说明文档                  <br>
> ├── .vscode/          // vscode 配置文件               <br>
> ├── main.c            // 项目源文件                    <br>
> └── build/            // 编译生成文件（不纳入版本控制）  <br>

## 项目版本记录

- v1.0.0
    采用彩色俄罗斯方块设计，在 Windows 终端中运行。
    已发现的 Bug：
    - 为了显示俄罗斯方块下落，清除上一个位置的方块并绘制下一个位置的方块时，会将除了该方块以外的，原本在其周围的方块擦除，导致该方块与周围的方块串色。
    - 满行消除并将上方方块下移时，会导致上方方块错误显示，即在原本没有方块的位置显示方块。

- v2.0.0
    采用单色俄罗斯方块设计，在 Windows 终端中运行。

- v3.0.0
    提供单色和彩色俄罗斯方块，可以在设置中修改，在 Windows 终端中运行。
